# 井上家 こだわり白湯

愛知県知多市にあるラーメン店「井上家」のホームページです。

## サイト概要

- 看板メニュー: 鶏白湯ラーメン
- 住所: 愛知県知多市にしの台1丁目2606
- 営業時間: 11:00〜15:00 / 17:00〜21:00
- 定休日: 月曜日

## 構成

- `index.html` - ページ本体
- `style.css` - スタイル
- `script.js` - ハンバーガーメニュー・スクロールアニメーション

## 編集について

メニューの価格・電話番号などはダミー値です。実際の内容に合わせて `index.html` を編集してください。
